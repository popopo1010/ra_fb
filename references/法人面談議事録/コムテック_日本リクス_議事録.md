## 7. コムテック側の採用方針

### 特色

- 「ディクス／リクス経由の流れだけ」ではなく、**コムテック独自の特色**も打ち出したい

### 募集イメージ

| 項目 | 内容 |
|------|------|
| 今年度 | 5名は採用計画に織り込み済み（人がいれば追加で取りたい） |
| 案件 | 中小規模中心。断っている案件もあるため人員増で受注拡大余地 |
| 目標内訳 | **実工事（現場作業）3名＋施工管理寄り2名**のイメージ |
| 教育 | 1対1のOJT体制あり。**ポテンシャルがあれば未経験も育成可能** |
| 人材構成 | 経験者／未経験者は社内で概ね半々の印象 |

### 事業のPRポイント

- オフィス内工事が多く、**暑さ寒さなど気候影響を受けにくい**
- 既存稼働中オフィスでの移設・設定変更等が多く、全くの新規工事よりも**"運用中の環境での工事"が中心**

### 年収レンジ感（コムテック）

- 主レンジ：**400〜500/600万程度**（高度経験者は必須ではない前提）
