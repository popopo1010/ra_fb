- **20代後半〜30代前半**がボリュームゾーン

### 経験の評価

| 経験 | 評価 |
|------|------|
| ゼネコン経験者 | 高く評価 |
| サブコン経験のみ | やや厳しめだが内容次第で検討可 |
| 派遣会社経由の施工管理経験 | 内容次第で検討対象 |
| 新築経験が多い方 | より評価 |
| 改修のみ | 相談可 |
| 前職の会社（競合・ライバル） | 特に不問 |

- RC造／S造いずれも可

## 年収レンジ（目安）

### 内定が出やすい年収帯

- **530万円〜730万円**（施工管理）
- **年収上限の目安**：750万円程度（管理職採用でない場合はそれ以上は難しい）

### 経験別イメージ（参考）

| 経験年数 | 年収目安 |
|----------|----------|
| 5年程度 | 〜600万円前後 |
| 10年程度 | 〜660万円前後 |
| 700万円超 | 経験・役割次第で個別判断 |

## 資格に関する考え方

- 資格なしでも施工管理経験があれば検討可
- 1級・2級建築施工管理技士、技士補は評価対象
- 資格取得予定者も経験内容次第で検討
