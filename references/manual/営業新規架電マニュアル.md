# XCHANGEの強み（差別化ポイント）

## 🏆 30秒で伝える差別化ピッチ

```
「当社の強みは3つあります。

1つ目は、施工管理経験者が候補者の紹介をしているので、
現場のことを理解したマッチングができます。
良いことも悪いことも全部お伝えするので、入社後のギャップを防げます。

2つ目は、候補者と1年後、3年後、10年後のキャリアまで
一緒に設計してからご紹介するので、条件だけのマッチングになりません。

3つ目は、入社後も候補者と面談して、齟齬があれば御社にフィードバックし、
定着をサポートします。実際、創業から3年以内に辞めた方は1人もいません。」
```

## 📝 差別化ポイント詳細

| 強み | 詳細 | 相手に響くポイント |
| --- | --- | --- |
| **ドメイン知識** | 施工管理経験者が紹介担当 | 「現場のことがわかってない紹介会社」への不満を解消 |
| **キャリア設計** | 1年・3年・10年後まで候補者と話す | 条件マッチだけでなく、志向性マッチができる |
| **期待値調整** | 良いことも悪いことも全部伝える | 入社後の「聞いてない」を防ぐ |
| **定着率** | 創業から3年以内の退職者ゼロ | 「すぐ辞められる」リスクへの不安を解消 |
| **入社後フォロー** | 入社後面談＋法人へのフィードバック | 紹介して終わりではないサポート体制 |
| **集客力** | デジタルマーケで数十億運用経験のあるメンバーが立ち上げ | 紹介できる候補者の母数が多い |

### 🧠 心理学Tips：差別化の伝え方

> 心理学原理：具体性の法則
「定着率が高いです」→ 抽象的で響かない
「創業から3年以内に辞めた人は1人もいません」→ 具体的で信頼できる
> 
> 
> 数字と事実で語ることで、信頼性が格段に上がる。
>

# 必須の業界知識

## ⚡ 電気工事士と施工管理の違い【最重要】

| 項目 | 電気工事士 | 施工管理 |
| --- | --- | --- |
| **仕事内容** | 実際に手を動かして電気工事をする「職人」 | 現場全体を管理・指揮する「管理者」 |
| **必要資格** | 電気工事士1種・2種 | 電気工事施工管理技士1級・2級 |
| **キャリアパス** | 職人→親方 | 現場監督→所長→部長 |
| **年収レンジ** | 350〜500万 | 500〜800万 |

### ⚠️ 絶対に間違えてはいけないポイント

```
❌ 「電気工事士の経験者」と「施工管理の経験者」を混同する
❌ 「資格あり」＝「経験あり」と思い込む

✅ 電気工事士の資格を持っている ≠ 施工管理の経験がある
✅ 「電気工事士の資格持ち × 施工管理未経験」は採用ニーズがある！

```

> 🔥 これを理解していないと、相手との会話がずれて信頼を失う
>

# 営業マインドセット

## 🧠 心理学的大原則

### 1. 「売り込み」ではなく「問題解決」である

あなたは「人材を売りつける営業」ではなく、「採用課題を解決するパートナー」である。
この認識の違いが、声のトーン、言葉選び、すべてに影響する。

> 🔬 心理学的根拠：フレーミング効果
同じ内容でも「売り込み」と捉えるか「問題解決」と捉えるかで、相手の受け取り方が180度変わる。
「紹介させてください」→ 売り込み感
「御社の採用課題を一緒に解決したい」→ パートナー感
> 

### 2. 「断られる」のはあなたではなく「タイミング」

100件かけて5件OKが取れれば目標達成。95件は断られる前提。
断られても「今じゃなかっただけ」と切り替える。

### 3. 相手の「時間を奪う」意識を持つ

だからこそ、**短く・的確に・価値を伝える**。
長々と話す営業は嫌われる。

### 4. ドメイン知識で信頼を勝ち取る

**成功する営業の共通点：相手の事業・戦略を理解している**

電気工事と施工管理の違い、年収相場、資格体系を理解していないと、
会話がずれて信頼を失う。逆に理解していれば「この人はわかってる」と信頼される。

# Phase 1：受付突破

## 🎯 このフェーズのゴール

人事担当者に電話を繋いでもらう

## 📝 基本スクリプト

```
「お忙しいところ失礼いたします。
電気工事士で20〜30代の方、7〜8名ほどご紹介できる方がおりまして、
いま人事ご担当の方いらっしゃいますでしょうか？」

```

### 🧠 心理学Tips：なぜこのスクリプトが効くのか

| 要素 | 心理学原理 | 解説 |
| --- | --- | --- |
| 「7〜8名」という具体的数字 | **具体性の法則** | 「たくさん」より「7〜8名」の方が信頼性が高く感じる |
| 「20〜30代」という年齢限定 | **希少性の原理** | 若手は市場で希少。「誰でもいます」より価値を感じさせる |
| 「ご紹介できる方がおりまして」 | **プッシュではなくプル** | 「紹介させてください」（押し売り）ではなく、「紹介できる人がいる」（情報提供）の姿勢 |

---

## 🚫 断られた時の対応

### パターン①「いま担当者がいない」

**スクリプト：**

```
「承知いたしました。大体何時頃お戻りになりますか？
その時間に改めてお電話させていただきます。」

```

**アクション：** リストに🔥マークをつけて後追い対象にする

> 🧠 心理学Tips：一貫性の原理
「何時に戻る」と教えてくれた時点で、相手は心理的に「その時間に電話が来る」ことを受け入れている。
再度かけた時に断りにくくなる。
> 

---

### パターン②「新規お断り」（主に大手）

**判断基準：**

- 大手企業 → お問い合わせフォーム/メール対応に切り替え
- 中堅以下 → 粘る価値あり

**粘る場合のスクリプト：**

```
「承知いたしました。ただ、かなり転職意欲の高い方がおりまして、
履歴書だけでも拝見いただくことは可能でしょうか？
必要な人材がいればピックアップいただくだけで構いません。
お送りするのは無料ですし、職務経歴書だけでもご覧いただけませんか？」

```

### 🧠 心理学Tips：なぜこのスクリプトが効くのか

| 要素 | 心理学原理 | 解説 |
| --- | --- | --- |
| 「履歴書だけでも」 | **フット・イン・ザ・ドア** | 小さな要求（見るだけ）から入り、大きな要求（契約）に繋げる |
| 「お送りするのは無料」 | **返報性の原理** | 無料で価値あるものを提供すると、お返しをしたくなる心理 |
| 「ピックアップいただくだけ」 | **労力の最小化** | 相手の負担を極限まで下げることで心理的ハードルを下げる |

**アクション：** 履歴書送付後、3営業日後にフォロー電話

# Phase 2：人事担当接続〜ニーズ把握

## 🎯 このフェーズのゴール

- 電気工事 or 施工管理、どちらにニーズがあるか特定
- 年齢層・経験レベルの優先順位を把握
- 話を聞く体制を作る

## 📝 基本スクリプト

### Step 1：再度の自己紹介＆ニーズ確認

```
「お忙しいところ失礼いたします。〇〇会社の△△と申します。
（該当都道府県）で、20〜30代の電気工事士・施工管理の方が
7〜8名ほど転職を希望されておりまして、ご紹介させていただきたく
お電話いたしました。

早速ですが、電気工事と施工管理でしたら、
どちらの方がニーズございますか？」

```

### 🧠 心理学Tips：二者択一質問の威力

> 心理学原理：選択肢の限定（Alternative Close）
「必要ですか？」（Yes/No）ではなく「どちらが必要ですか？」（A or B）と聞く。
「必要ない」という選択肢を無意識に排除させる。
> 

---

### Step 2：年齢層・経験レベルの確認

**相手：「施工管理ですね」と回答後**

```
「ありがとうございます。施工管理ですね。
20〜30代のポテンシャル層と、40代のバリバリ即戦力層、
どちらの方がニーズありますか？」

```

### 🧠 心理学Tips

| 要素 | 心理学原理 | 解説 |
| --- | --- | --- |
| 「ポテンシャル」「バリバリ即戦力」 | **ラベリング効果** | 年齢ではなく価値でラベルをつけることで、「どちらも魅力的」と感じさせる |
| 再度の二者択一 | **一貫性の原理** | 一度「施工管理」と答えた相手は、次の質問にも答えやすくなる |

---

### Step 3：ポテンシャル層の深掘り

**相手：「20〜30代ですね」と回答後**

```
「ありがとうございます。
20〜30代で資格をお持ちの方はもちろんですが、
職人としてバリバリやっていて、電気の現場経験はあるけど、
これから施工管理をやりたいという方がいるんですが、いかがでしょう？」

```

### 🧠 心理学Tips：条件の段階的緩和

> 心理学原理：ドア・イン・ザ・フェイス（譲歩の返報性）
最初に「資格あり」で期待を持たせ、次に「経験あり・資格なし」を提案。
「資格なしでもいいか」という譲歩を引き出す。
> 

---

### ⚠️ 【重要】ニーズ確認時の注意点

```
「電気工事士の資格あり」と「施工管理の経験あり」を混同しない！

❌ 相手：「経験者がほしい」
   自分：「電気工事士の経験者ですね」
   → 施工管理の経験者のことかもしれない！

✅ 相手：「経験者がほしい」
   自分：「施工管理の経験者ということでしょうか？
        それとも電気工事の現場経験がある方でしょうか？」
   → 明確に確認する

```

> 🔥 ここがずれると、その後の会話が全部ずれる！必ず確認！
>

# Phase 3：ヒアリング深掘り

## 🎯 このフェーズのゴール

- 求める人材像を具体化する
- 採用条件を把握する
- **候補者を動機付けできる情報**を収集する

## 📝 ヒアリング項目チェックリスト

| # | 項目 | 確認すべきこと | 質問例 |
| --- | --- | --- | --- |
| 1 | 事業内容 | メインの案件は何か | 「HPを拝見したのですが、実際に一番多く受けている案件はどういったものですか？」 |
| 2 | 募集エリア | どの地域で人が必要か | 「エリアでいうと、一番募集されているのはどちらですか？」 |
| 3 | 経験者年収 | 相場感の把握 | 「施工管理10年、30代中盤だと、どれくらいでスタートできますか？500〜600万くらいですか？」 |
| 4 | 未経験者年収 | ポテンシャル層の条件 | 「未経験の方だと、どれくらいの年収からスタートですか？」 |
| 5 | キャリアパス | 3〜5年後の姿 | 「3年後に主任になられた場合、どれくらいの年収になりますか？」 |
| 6 | 主任の仕事内容 | 具体的な業務 | 「主任になったら、どういった仕事を任せたいですか？」 |
| 7 | 残業時間 | ワークライフバランス | 「残業は大体30時間くらいですか？」 |
| 8 | 休日 | 週休二日か、土日休みか | 「お休みは週休二日ですか？土日休みですか？」 |
| 9 | 出張 | 頻度と期間 | 「出張はありますか？どの程度行かれますか？」 |

---

## 📝 会話の流れスクリプト

### Step 1：事業内容の確認（HP見ながら）

```
「話しながらで恐縮ですが、御社のHP拝見しています。
実際のところ、一番多く受けている案件はどういったものですか？」

```

### 🧠 心理学Tips

| 要素 | 心理学原理 | 解説 |
| --- | --- | --- |
| 「HP拝見しています」 | **ラポール構築** | 事前準備している＝本気で関わろうとしている印象を与える |
| 「実際のところ」 | **本音を引き出すトリガー** | 建前ではなく本音を聞きたい姿勢を示す |

---

### Step 2：相手を褒める（信頼構築の鍵）

```
「御社、〇〇エリアで〇〇の案件を多くやられているんですね。
すごいですね、あの規模の案件を手掛けられる会社は多くないですよね。」

```

### 🧠 心理学Tips：なぜ「褒める」が重要か

> 心理学原理：好意の返報性
人は自分を認めてくれる人に好意を持つ。
相手の事業・戦略を理解した上で具体的に褒めると、
「この人はわかってる」「この人と話したい」という気持ちになる。
> 
> 
> **成功する営業の共通点：相手を褒められる材料を持っている**
> 

---

### Step 3：年収ヒアリング（アンカリングを使う）

```
「経験者の方をご紹介する場合ですが、例えば施工管理10年で、
30代中盤の方だと、どれくらいでスタートできますか？
500〜600万くらいでしょうか？」

```

### 🧠 心理学Tips：アンカリング効果

> 心理学原理：アンカリング
先に「500〜600万」という数字を出すことで、相手の回答をその近辺に誘導する。
何も言わないと「400万から」と言われる可能性があるが、
アンカーを置くことで相場を引き上げる。
> 

---

### Step 4：キャリアパスの確認（動機付け材料の収集）

```
「3年後に主任になられた場合、どれくらいの年収になりますか？」

「主任になったら、どういった仕事を任せたいとお考えですか？」

```

**このあとの重要なセリフ：**

```
「なぜこういったことをお聞きしているかと言いますと、
入社後にどういった働き方ができるかを候補者にお伝えすることで、
しっかり動機付けができて、御社に本当に必要な人材を
ご紹介できると考えているからです。」

```

### 🧠 心理学Tips：意図の開示

| 要素 | 心理学原理 | 解説 |
| --- | --- | --- |
| 「なぜ聞いているか」を説明 | **透明性の原理** | 意図を開示することで、警戒心を解く。「何を企んでいるんだ？」という疑念を払拭 |
| 「御社に本当に必要な人材を」 | **Win-Win姿勢の提示** | 自社の利益だけでなく、相手の利益を考えている姿勢を示す |
| 「動機付け」という専門用語 | **専門性の演出** | プロとして採用を理解している印象を与える |

---

### Step 5：労働条件の確認

```
「残業は大体どれくらいですか？30時間くらいでしょうか？」

「お休みは週休二日ですか？」

「出張はありますか？ある場合、どの程度の頻度で行かれますか？」

```

### 🧠 心理学Tips：確認型質問

> 「残業はどれくらいですか？」（オープン）ではなく
「30時間くらいですか？」（確認型）で聞く。
> 
> 
> **効果：**
> 
> 1. 相手が答えやすい（Yes/Noベースで答えられる）
> 2. 会話のテンポが上がる
> 3. こちらが業界相場を知っている印象を与える

# Phase 4：料率交渉

## 🎯 このフェーズのゴール

- 基本料率35%の合意
- 経験者マッチ時の40〜50%交渉の布石を打つ
- 履歴書送付の約束を取り付ける

## 📝 基本スクリプト

### Step 1：35%の提示

```
「料率のお話なんですが、基本的には35%でお願いしております。
他社さんでも同じくらいかと思いますが、いかがでしょうか？」

```

### 🧠 心理学Tips

| 要素 | 心理学原理 | 解説 |
| --- | --- | --- |
| 「他社さんでも同じくらい」 | **社会的証明** | 「みんなやっている」ことで、35%が相場であるという認識を植え付ける |
| 「かと思いますが」 | **ソフトな断言** | 押し付けではなく、確認の姿勢で柔らかく |

---

### Step 2：経験者の場合の40〜50%交渉

```
「ちなみに、御社の求める人材にバッチリはまる方、
例えば施工管理でしっかり経験を積まれている方の場合は、
40〜50%でお願いすることもあるのですが、いかがでしょうか？」

```

**相手：「経験によるよね」と回答した場合**

```
「おっしゃる通りです。ですので、バリバリやられていて、
御社の求める経験にピッタリ合う方がいた場合は、
機会損失にならないよう、条件の交渉をさせていただくことは
可能でしょうか？」

```

### 🧠 心理学Tips：損失回避バイアス

> 心理学原理：損失回避（Loss Aversion）
人は「得をする」より「損をしない」ことに2倍強く反応する。
> 
> 
> 「いい人材を紹介します」→ 響きにくい
> 「機会損失にならないように」→ 「逃したくない」心理が働く
> 

---

### Step 3：厳しいと言われた場合

```
「承知いたしました。それでは、まずは35%をベースにさせていただき、
本当にバッチリ合う方がいた際には、改めてご相談させてください。
まずは履歴書をお送りしてもよろしいでしょうか？」

```

### 🧠 心理学Tips：一旦引いて再提案

> 心理学原理：ドア・イン・ザ・フェイス
一度「厳しい」と言われたら引く。しかし「改めてご相談」という布石は残す。
そして話題を「履歴書送付」という小さなYesに切り替える。
>

# 反論対応集

## 🔥 よくある断り文句と切り返し

### 1. 「すでに取引先がある」

```
「そうでしたか。失礼いたしました。
ちなみに、今お取引されている会社さんでカバーしきれていない
エリアや職種はございませんか？
当社は電気工事・施工管理に特化しているので、
補完的にお役に立てる部分があるかもしれません。」

```

**心理学：ニッチポジショニング**
全部を取ろうとせず、隙間を狙う姿勢を見せる。

---

### 2. 「今は採用していない」

```
「承知いたしました。
ちなみに、次に採用を再開されるのはいつ頃のご予定ですか？
そのタイミングでご連絡させていただいてもよろしいでしょうか？」

```

**心理学：コミットメントの先取り**
「いつ頃」を聞くことで、将来の接点を約束させる。

---

### 3. 「資料送っておいて」（断り文句として）

```
「承知いたしました。では、会社案内と併せて、
今すぐご紹介可能な方の職務経歴書を2〜3名分お送りしてもよろしいですか？
具体的な人材を見ていただいた方が、イメージしやすいかと思います。」

```

**心理学：具体化による興味喚起**
抽象的な「資料」ではなく、具体的な「職務経歴書」を送る。

---

### 4. 「人事じゃなくて現場で決める」

```
「なるほど、現場の方がご判断されるんですね。
でしたら、現場責任者の方にお繋ぎいただくことは可能でしょうか？
技術的なお話ができる方と直接お話しさせていただいた方が、
ミスマッチのないご紹介ができると思います。」

```

**心理学：意思決定者へのルート変更**
決裁者が違うなら、正しい相手に繋いでもらう。

---

### 5. 「料率が高い」

```
「おっしゃる通り、安くはないと思います。
ただ、当社は施工管理経験者が紹介を担当しているので、
ミスマッチによる早期退職のリスクを大幅に抑えられます。
実際、創業から3年以内に辞められた方は1人もいません。
採用コストをトータルで考えると、結果的にコスパが良いと
ご評価いただくことが多いです。」

```

**心理学：フレーミング効果**
「高い」→「トータルコストで安い」に視点を変える。

# 成功事例・失敗事例

## ✅ 成功パターン

### 成功事例①：相手を褒められた時

**状況：** HPで相手企業が大規模プラントの電気工事を手掛けていることを発見

**トーク：**

```
「御社、〇〇プラントの案件を手掛けられているんですね。
あの規模の電気工事を請けられる会社は限られていますよね。
現場の管理体制もしっかりされているんだと思います。」

```

**結果：** 相手が「そうなんですよ、うちはそこが強みで...」と話し始め、関係構築に成功

> 🔑 成功のポイント
> 
> - 事前にHPを見て、具体的な褒めポイントを準備していた
> - 業界知識があるからこそ「あの規模は難しい」と言えた
> - 相手の自尊心をくすぐり、話を引き出した

---

### 成功事例②：ドメイン知識で信頼を勝ち取った時

**状況：** 相手が「施工管理の経験者がほしい」と言った

**トーク：**

```
「施工管理経験者ですね。ちなみに、1級施工管理技士をお持ちの方が
理想でしょうか？それとも2級でも現場経験が豊富であればOKでしょうか？

あと、電気工事士の資格持ちで、これから施工管理にキャリアチェンジ
したいという方もいるんですが、そういった方はいかがですか？」

```

**結果：** 「資格の違いまでわかってるんだ」と信頼され、詳しいニーズを教えてもらえた

> 🔑 成功のポイント
> 
> - 資格体系（1級/2級）を理解していた
> - 「電気工事士資格持ち×施工管理未経験」という狙い目を提案できた
> - 「この人はわかってる」という信頼を獲得

---

## ❌ 失敗パターン

### 失敗事例①：ドメイン知識のずれ

**状況：** 相手が「経験者がほしい」と言った

**自分の認識：** 「電気工事士の経験者」のことだと思った
**相手の認識：** 「施工管理の経験者」のことだった

**結果：**

- 電気工事の職人さんを紹介しようとして話がずれた
- 「この人わかってないな」と思われて話が途絶えた

> 🔑 失敗の原因
> 
> - 「経験者」が何の経験者かを確認しなかった
> - 電気工事士と施工管理の違いを理解していなかった
> 
> **💡 対策**
> 「経験者」と言われたら必ず確認する：
> 「施工管理のご経験でしょうか？電気工事のご経験でしょうか？」
> 

---

### 失敗事例②：「資格あり」と「経験あり」の混同

**状況：** 相手が「電気工事士の資格持ちで、施工管理未経験の人はいる？」と聞いた

**自分の認識：** 「資格あり未経験」は求人としてないと思い込んでいた

**結果：**

- 「そういう方は難しいですね...」と答えてしまった
- 実際は「電気工事士資格持ち×施工管理未経験」は需要がある
- チャンスを逃した

> 🔑 失敗の原因
> 
> - 「電気工事士の資格持ち × 施工管理未経験」というパターンを知らなかった
> 
> **💡 対策**
> このパターンは需要があることを覚えておく！
> 職人経験＋電気知識があれば、施工管理デビューできる人材として価値がある
>

## 📅 タイミング

```
「お忙しいところ失礼いたします。先日履歴書をお送りした〇〇会社の△△です。
お送りした資料、ご確認いただけましたでしょうか？」

```

**相手：「まだ見てない」**

〇〇様（候補者）は今週中にも他社の選考が進む予定でして、

> 「他社の選考が進む」→ この人材を逃すかもしれない
「今週中に」→ 時間的制約を設ける
人は「なくなるかもしれない」ものに反応する
>

# NGワード・NG行動

## 🚫 絶対に言ってはいけないこと

| NGワード | なぜダメか | 代替表現 |
| --- | --- | --- |
| 「誰でもいます」 | 価値がないように聞こえる | 「〇〇のスキルを持った方がいます」 |
| 「安くします」 | 安売り感＝質が低い印象 | 「条件はご相談させてください」 |
| 「他社より絶対いい」 | 根拠のない比較は信用を失う | 「当社の強みは〇〇です」 |
| 「採用しないと損」 | 押し売り感が強すぎる | 「機会損失にならないように」 |
| 「今すぐ決めてください」 | プレッシャーは逆効果 | 「いつ頃ご回答いただけますか？」 |

## 🚫 NG行動

| NG行動 | なぜダメか | 正しい行動 |
| --- | --- | --- |
| 一方的に話し続ける | 相手は聞いていない | 質問→回答→質問のリズムで会話 |
| 断られて食い下がりすぎ | 印象が悪くなる | 2回断られたら引く、次の機会を狙う |
| HPを見ずに電話 | 準備不足がバレる | 最低限、事業内容は把握しておく |
| メモを取らない | 同じことを聞くことになる | 必ず記録、CRMに即入力 |
| 約束した時間に電話しない | 信頼を失う | リマインダー設定、絶対に守る |
| 「経験者」を確認せず進める | 話がずれて信頼を失う | 電気工事か施工管理か必ず確認 |

# 📚 付録：心理学用語集

| 用語 | 意味 | 営業での活用 |
| --- | --- | --- |
| **フット・イン・ザ・ドア** | 小さな要求を通すと、大きな要求も通りやすくなる | 「履歴書だけでも見て」→「面談しませんか」 |
| **ドア・イン・ザ・フェイス** | 大きな要求を断らせてから、小さな要求を通す | 「50%で」→「35%でいかがですか」 |
| **社会的証明** | 「みんながやっている」と安心する | 「他社さんでも35%~50%です」 |
| **希少性の原理** | 手に入りにくいものは価値が高く感じる | 「この候補者は今週中に他社選考が...」 |
| **損失回避** | 人は得するより損を避けたい | 「機会損失にならないように」 |
| **アンカリング** | 最初に提示された数字が基準になる | 「500〜600万くらいですか？」 |
| **返報性の原理** | 何かをもらったらお返ししたくなる | 「無料で履歴書をお送りします」 |
| **一貫性の原理** | 一度言ったことと矛盾したくない | 小さなYesを積み重ねる |
| **ラポール** | 信頼関係、親密さ | 「HP拝見してます」「なぜ聞くかというと...」 |
| **好意の返報性** | 認めてくれる人に好意を持つ | 相手の事業・戦略を褒める |

# 🎯 最後に：営業の本質

> 「売り込む」のではなく「課題を解決する」「断られる」のではなく「タイミングを待つ」「話す」のではなく「聞き出す」「知識がない」のではなく「プロとして理解している」

この4つを忘れなければ、必ず成果は出る。
