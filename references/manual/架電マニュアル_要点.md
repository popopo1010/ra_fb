# 架電マニュアル 要点（FB参照用）

> Phase 1 受付突破 + Phase 2 ニーズ把握 + Phase 3 ヒアリング。架電FB評価の基準。

---

## Phase 1：受付突破（目標30秒以内）

### 基本スクリプト

```
「お忙しいところ失礼いたします。
電気工事士で20〜30代の方、7〜8名ほどご紹介できる方がおりまして、
いま人事ご担当の方いらっしゃいますでしょうか？」
```

### ポイント

- 「7〜8名」など具体的数字で信頼性
- 「20〜30代」で希少性
- 「ご紹介できる方がおりまして」＝プル型（押し売りではない）

### 断られた時

- 担当不在→「何時頃お戻りになりますか？」で再架電の約束
- 新規お断り→「履歴書だけでも拝見いただけませんか？」（フット・イン・ザ・ドア）

---

## Phase 2：人事担当接続〜ニーズ把握（目標2分以内）

### 二者択一で聞く

```
「電気工事と施工管理でしたら、どちらの方がニーズございますか？」
「20〜30代のポテンシャル層と、40代のバリバリ即戦力層、どちらがニーズありますか？」
```

### 重要：混同しない

- 「電気工事士の資格あり」≠「施工管理の経験あり」
- 相手が「経験者」と言ったら、電気工事の現場経験か施工管理経験かを必ず確認

---

## Phase 3：ヒアリング深掘り（目標5〜10分）

### ヒアリング項目チェックリスト

| # | 項目 | 確認すべきこと |
|---|------|----------------|
| 1 | 事業内容 | メインの案件は何か |
| 2 | 募集エリア | どの地域で人が必要か |
| 3 | 経験者年収 | 相場感（500〜600万等） |
| 4 | 未経験者年収 | ポテンシャル層の条件 |
| 5 | キャリアパス | 3〜5年後の姿、主任の仕事内容 |
| 6 | 残業時間 | 月30時間程度か |
| 7 | 休日 | 週休二日か、土日休みか |
| 8 | 出張 | 頻度と期間 |

### 確認型質問

「残業はどれくらいですか？」より「30時間くらいですか？」の方が答えやすい。相場を知っている印象も与える。

### 動機付け材料の収集

「3年後に主任になったらどれくらいの年収に？」「主任になったらどういった仕事を任せたい？」→ 候補者への伝え方に活用。

---

## 必須の業界知識

- 電気工事士＝職人、施工管理＝管理者
- 電気工事士資格持ち×施工管理未経験＝採用ニーズが高い層
- 年収相場：電気工事士2種 350〜400万、1種 400〜500万、施工管理 500〜800万
