# 営業のヒアリング情報

営業ヒアリングで収集した情報や、商談で活用するフレームワーク・構造マップを格納するフォルダです。

## 用途

- 商談前の事前リサーチ・市場理解
- 法人ヒアリング時の質問設計
- 求職者・法人への訴求ポイント整理
- 資格別年収・年齢制約などの前提情報の共有

## 格納ファイル

| ファイル | 内容 |
|----------|------|
| **電気施工管理_構造マップ_v4.html** | 電気施工管理特化版の構造マップ（ブラウザで開く） |

### 構造マップの構成

1. **経営課題ツリー** - 市場環境→事業構造→案件→組織→将来の5層
2. **採用課題ツリー** - 採れない/辞める/育たない＋資格別年収・年齢制約
3. **経営ループ** - 市場環境→商流→案件→人材充足のサイクル
4. **採用ループ** - 採用チャネル→好循環/悪循環の分岐
5. **前提・訴求** - 紹介前提・求職者訴求・法人訴求・市場成長ドライバー

## パス定数

Python からは `ra_fb.config.HEARING_DIR` で参照可能。
