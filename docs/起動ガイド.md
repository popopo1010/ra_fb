# RA FB システム 起動ガイド

## 共通：初回セットアップ

```bash
git clone https://github.com/popopo1010/ra_fb.git
cd ra_fb
pip install -r requirements.txt
cp .env.example .env
# .env を編集して API キー・Webhook URL を設定
```

---

## ikeobook15 の起動

```bash
cd /Users/ikeobook15/RA_FBシステム
python scripts/slack_server.py
```

venv 使用時:
```bash
source venv/bin/activate
python scripts/slack_server.py
```

---

## kazushi の起動

```bash
cd /Users/kazushi/ra_fb
git pull origin main
source venv/bin/activate
python scripts/slack_server.py
```

※ `python` が見つからない場合は `python3` または `./venv/bin/python scripts/slack_server.py` を使用

---

## 起動コマンド一覧

| 用途 | コマンド |
|------|----------|
| Slack /rafb_call, /rafb_mtg | `python scripts/slack_server.py` |
| Notta Webhook | `python scripts/webhook_server.py` |
| CLI（RA FB） | `python scripts/cli.py ra <ファイル>` |
| CLI（CA FB） | `python scripts/cli.py ca <ファイル>` |
| 後方互換 | `python scripts/generate_feedback.py <ファイル>` → cli.py ra と同じ |
| 後方互換 | `python scripts/slack_app_server.py` → slack_server.py と同じ |

---

## トラブルシューティング

| 現象 | 対処 |
|------|------|
| `No such file or directory: slack_server.py` | `git pull origin main` で最新を取得 |
| `command not found: python` | `source venv/bin/activate` または `python3` を使用 |
| `TypeError: unsupported operand type(s) for \|` | Python 3.10 未満の場合。`git pull` で Python 3.9 互換版を取得 |
| `slack_bolt がインストールされていません` | `pip install -r requirements.txt` |
