## .env の設定

| 変数 | 必須 | 状態 | 備考 |
|------|------|------|------|
| `SLACK_WEBHOOK_URL` | ○ | ✅ 設定済み | #dk_ra_初回架電fb の Incoming Webhook |
| `SLACK_APP_TOKEN` | Slack App 利用時 | ✅ 設定済み | xapp- で始まる App-Level Token |
| `SLACK_BOT_TOKEN` | Slack App 利用時 | ❌ **未設定** | xoxb- で始まる Bot User OAuth Token |
| `ANTHROPIC_API_KEY` | オプション | 未設定 | AI フィードバック用（未設定時はテンプレート） |

### 修正が必要

**SLACK_BOT_TOKEN** が未設定です。Slack App（/rafb）を使う場合は必須です。

1. [api.slack.com/apps](https://api.slack.com/apps) → 対象 App
2. **OAuth & Permissions** → **Bot User OAuth Token** をコピー
3. `.env` に追加: `SLACK_BOT_TOKEN=xoxb-xxxxxxxxxxxx`
