## サンプル1：電巧社（設備部・工事部）

```
📞 初回架電FB | DENKOSHA_電巧社 | RA: 茂野

【良かった点】
・冒頭で「電気工事士と施工管理の資格持ちで20〜30代が複数名」と具体的な数字を伝え、受付突破できた
・設備部と工事部の違い（未経験OK枠 vs 即戦力枠）を整理してヒアリングできた
・年収レンジ（400〜500万）と転職回数の目安を明確に聞けた
・直行直帰OK、残業17h程度など働き方を具体的に確認した

【改善点】
・工事部の「1種電工必須」を最初に確認しておくと、紹介時のミスマッチを防げる
・採用ペース（毎月1〜2名）と期限（今期中）をセットで聞くと、緊急度が伝わりやすい

【採用概要状況】
・採用必要数：設備部4名、工事部4名（計8名）。毎月1〜2名ずつ入社、今期中に採用完了
・エリア：東京、名古屋、大阪、広島、福岡（★東京・名古屋・福岡が優先）
・必要資格：設備部は30代まで未経験OK。工事部は施工管理技士2級以上、第一種電気工事士必須
・未経験の採用有無：設備部は30代まで未経験OK。工事部は即戦力が中心
・年齢：社会人3年以上〜50代前半。設備部は30代までは未経験可、30代以降は施工管理経験者
・経験：設備部はドラッグストア等の決まった施工方式で経験積みやすい。工事部は警視庁・大型倉庫等の大規模案件
・年収（固定/賞与）：400〜500万。電気工事経験あれば500万。2種30代前後で500万近く。賞与は資格取得で等級アップ
・出張：工事部は大型案件で出張多め。大規模案件は出張がネックになる場合あり
・採用において重視している点：定着への懸念がないか、転職理由の一貫性、関心・意欲、コミュニケーション力。転職回数は20代3社目まで、30代はもう1社程度（経験あれば4社目もOK）
・その他：東芝パートナー97年、健康経営ブライト500。1名採用で約1億円売上増（設備部）。工事部は現場負荷増大・離職リスクが課題

【進めるにあたっての障壁】
・転職回数2回超えると厳しい（経験あれば4社目でもOK）
・口コミスコア低め（OpenWork 2.9）→評価制度・昇進基準に課題感。候補者への説明が必要
・東芝依存度が高い点を懸念する候補者もいる可能性

【具体的に聞き方を変えた方がいい点と言い回し】
・資格の確認：「設備部と工事部で要件が違うと思うのですが、工事部は1種電工が必須でしょうか？」
・年収の深掘り：「経験者で500万いく方は、何年目くらいの方が多いでしょうか？」
・転職回数：「転職回数はどのくらいまでであればご検討いただけますか？」

【全体所感】
設備部・工事部の両方で採用ニーズがあり、エリアも複数。未経験OK枠（設備部）と即戦力枠（工事部）を分けて紹介するとマッチしやすい。転職回数・年収の目安を押さえておくことが重要。候補者には「創業97年」「1名で1億円売上増」など具体的な数字で訴求できる。
```

## サンプル2：東海EC（名古屋・小牧）

```
📞 初回架電FB | TOKAI_EC_名古屋小牧 | RA: 重野

【良かった点】
・名古屋と小牧で採用要件が異なることを確認できた（名古屋は有資格者優先、小牧は若手なら資格なしでも可）
・若手の初年度年収（350万弱）を具体的に聞けた
・繁忙期（3月末まで）と残業実態（月60〜80hのケースあり）を確認した
・「まず1人ずつ」の採用方針を把握した

【改善点】
・経験者の年収上限（600万超は難しい）を早めに聞いておくと、紹介時のミスマッチを防げる
・出張・転勤の有無を確認しておくと良い

【採用概要状況】
・採用必要数：名古屋・小牧それぞれ「まず1人ずつ」。人数上限は設けていない
・エリア：名古屋、小牧（愛知県）
・必要資格：名古屋は施工管理技士等の有資格者を優先。小牧は若手であれば電気工事士資格なしでも書類選考対象
・未経験の採用有無：若手（20代）は資格がなくても意欲があれば面接で判断。20代前半・未経験で入社し2年で2級取得した社員の実績あり
・年齢：20代〜40代前半が中心。35歳以上は経験次第。40代後半も経験値次第で検討可
・経験：名古屋は公共・民間（商店・プラント）。小牧は物流設備・機械メーカー案件。即戦力は1級土木/電気/管工事施工管理技士
・年収（固定/賞与）：若手初年度350万弱（残業込み400万未満）。経験者500万超は要調整、600万前後が検討ライン。賞与は「寸止めレベル」
・出張：未確認
・採用において重視している点：年齢・経験値・業界への意欲・チャレンジ精神を総合判断。高年収希望はミスマッチになりやすい
・その他：創業70周年、引き合い多い。繁忙期は3月末まで。36協定遵守、月45h超は年6回まで。人材紹介は理論年収35%希望

【進めるにあたっての障壁】
・若手は年収350万弱で、高年収希望者はミスマッチ
・繁忙期の残業（月60〜80h）がネックになる候補者もいる
・中小企業のため年収600万超は難しい

【具体的に聞き方を変えた方がいい点と言い回し】
・年収の確認：「若手で入社された場合、初年度の年収はどのくらいが目安でしょうか？」
・資格の確認：「小牧の若手枠は、資格がなくても書類選考の対象になりますか？」
・採用ペース：「名古屋と小牧、それぞれ何人ずつ採用予定でしょうか？」

【全体所感】
名古屋は有資格者、小牧は若手（資格なし可）と拠点で要件が異なる。若手・ポテンシャル層の紹介がマッチしやすい。年収は現実的な水準であることを候補者に伝える必要がある。創業70周年・引き合い多いなど成長性は訴求ポイントになる。
```

## サンプル3：聞けていない項目が多い場合

```
📞 初回架電FB | SAMPLE_企業_001 | RA: 小山田

【良かった点】
・採用担当に繋がり、話を聞く機会を得られた
・「条件に合いそうな方を数名お送りできる」とメリットを伝えてからヒアリングに入った
・次のアクション（履歴書送付）を約束できた

【改善点】
・開き方に「〇名」「20〜30代」などの具体的な数字を入れると、受付突破率が上がりやすい
・接続後、相手のメリットを先に伝えてからヒアリングする流れは良い。継続して実施

【採用概要状況】
・採用必要数：未確認
・エリア：未確認
・必要資格：未確認
・未経験の採用有無：未確認
・年齢：未確認
・経験：未確認
・年収（固定/賞与）：未確認
・出張：未確認
・採用において重視している点：未確認
・その他：履歴書を送付予定。採用ニーズの有無は次回確認

【進めるにあたっての障壁】
・初回架電のため、詳細な採用条件は未確認
・担当者が多忙で、打ち合わせ日程は未定

【具体的に聞き方を変えた方がいい点と言い回し】
・受付向け：「電気工事士や施工管理の資格持ちで、20〜30代の方が複数名おりまして、御社の条件に合いそうな方をご紹介できればと思い、お電話しました。採用担当の方、お手すきでしょうか？」
・担当者接続後：「御社の条件をお聞きして、合いそうな方を数名、メールでお送りできる状態です。勤務地や職種など、条件を少しお聞かせいただけますか？」

【全体所感】
初回架電として受付突破・担当者接続・次のアクション獲得まで進んだ。次回は採用必要数・エリア・資格・年収・出張・重視点を体系的にヒアリングする。履歴書送付後、3営業日以内にフォロー電話を入れる。
```
