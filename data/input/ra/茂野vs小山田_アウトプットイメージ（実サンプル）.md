# 茂野 vs 小山田：アウトプットイメージ（実際のサンプル使用）

> 実際の架電記録の文字起こしを使い、Before（現状）と After（改善イメージ）を比較する。

---

## 1. 受付突破：開き方の比較

### シーン① サンケイビルマネジメント（小山田・実際）

**Before（実際の音声）**

```
受付：「株式会社サンケイビルマネジメントでございます。」

小山田：「あ、お忙しいところ恐れ入ります。私、エクスチェンジ株式会社の小山田と申します。
        お世話になっております。今回ですね、電気工事士と施工管理の人材紹介の件で
        ご連絡させていただいたんですが。はい。もしよろしければ、あの、採用を担当されている方と
        少しお話できできないかなと思いまして。」

受付：「かしこまりました。確認いたしますので、少々お待ちください。」
→ 繋がった
```

**After（改善イメージ）**

```
受付：「株式会社サンケイビルマネジメントでございます。」

小山田：「お忙しいところ恐れ入ります。エクスチェンジ株式会社の小山田と申します。
        電気工事士や施工管理の資格持ちで、20〜30代の方が複数名おりまして、
        御社の条件に合いそうな方をご紹介できればと思い、お電話しました。
        採用担当の方、お手すきでしょうか？」

受付：「かしこまりました。確認いたしますので、少々お待ちください。」
```

**差分のポイント**
- 「人材紹介の件」→「資格持ちで20〜30代が複数名、御社に合いそうな方をご紹介」
- 具体的な数字・用件を冒頭で伝え、取次する価値が明確に

### シーン② リーフテクニカ（小山田・実際）※社名確認が入ったケース

**Before（実際の音声）**

```
受付：「（電話を受ける）」

小山田：「あの、私エクスチェンジ株式会社の小山田と申します。お世話になります。
        お世話になります。今回ですね、電気工事士や施工管理の人材紹介の件で
        ご連絡したんですけれども。はい。もしよろしければ、採用担当の方と
        少しお話できないかなと思いまして。」

受付：「社名様も一度お伺いしてよろしいですか？」  ← 確認が入った

小山田：「すみません。エクスチェンジ株式会社と申します。」

受付：「はい、少々お待ちください。確認いたします。」
→ 繋がったが、確認のやり取りで時間がかかった
```

**After（改善イメージ）**

```
受付：「（電話を受ける）」

小山田：「お忙しいところ恐れ入ります。エクスチェンジ株式会社の小山田と申します。
        電気工事士や施工管理の資格持ちで、20〜30代の方が複数名おりまして、
        御社の条件に合いそうな方をご紹介できればと思い、お電話しました。
        採用担当の方、お手すきでしょうか？」

受付：「かしこまりました。少々お待ちください。」  ← 用件が明確なので確認が入りにくい
```

**差分のポイント**
- 用件が具体的なため、「社名は？」と聞き返されるリスクが減る
- 会社名＋用件＋依頼を一気に伝え、取次の判断をしやすくする

### シーン④ エアルエンジ（茂野・実際）※参考：繋がりやすい開き方

**茂野の実際の音声**

```
受付：「はい。お世話になります。」

茂野：「エクスチェンジ株式会社重野と申します。はい。僕あの、電気工事士と施工管理の資格持ちの
```

- 繋がった直後に「8名」と具体的な数字を出す

## 2. 接続後：担当者への最初の一言の比較

### シーン⑥ リーフテクニカ（小山田・実際）※相手が紹介料を先に聞いたケース

**Before（実際の音声）**

```
担当者：「（取次後）はい。」

小山田：「今回ですね、あの人材の紹介の件でご連絡いたしまして。はい。
        弊社の電気工事士や施工管理の二十代、三十代の方がですね、複数名動いておりまして。
        はい。もしよろしければ、御社の条件に合った方を数名、条件聞いた上でピックアップさせていただいて、
        一度メール等で送らせていただいて、あの、見ていただけないかなと思ってご連絡したんですけれども。
        はい。はい。もしよろしければそちらでですね、採用ニーズとかそういうものを
        少しお聞きさせていただく時間をいただけないかなと思いまして。」

担当者：「なるほどですね。えーと、紹介料はどれくらいになるんでしょうか？」  ← 相手が先に聞いた
```

**After（改善イメージ）**

```
担当者：「（取次後）はい。」

小山田：「お世話になっております。弊社、電気工事士や施工管理の20代・30代の方が複数名おりまして、
        御社の条件をお聞きして、合いそうな方を数名、メールでお送りできる状態です。
        もしよろしければ、勤務地や職種など、条件を少しお聞かせいただけますか？
        あと、紹介料は35〜50%という形になっておりますが、御社のご意向に合わせて調整も可能です。」
```

**差分のポイント**
- 「採用ニーズをお聞きしたい」が先 → 「合いそうな方を数名お送りできる状態」を先に
- 紹介料を相手が聞く前に、こちらから触れることで主導権を握る

### シーン⑦ サンケイビルマネジメント（小山田・実際）※接続後は比較的良いケース

**小山田の実際の音声（担当者接続後）**

```
担当者：「はい、お電話変わりました。採用担当でございます。」

小山田：「あ、お忙しいところ恐れ入ります。私、エクスチェンジ株式会社の小山田と申します。
        お世話になっております。あ、お世話になっております。今回ですね、電気工事士と施工管理の
        人材紹介の件でご連絡したんですけれども。はい。
        あの、弊社ですね、電気工事士や施工管理の二十代、三十代の方がですね、複数名動いておりまして。
        はい。御社の条件をお聞きして、そちらで合いそうな方を数名、今ご紹介できる状態なんですけれども。
        ありがとうございます。はい、もしよろしければ、少しお話をお伺いしてですね。はい。
        あの適した人材といいますか、そういう方をメール等で送らせていただき、
        一度送らせていただけないかなというところだったんですけれども。」
```

**ポイント**
- 接続後に「二十代、三十代が複数名」「条件に合いそうな方を数名、今ご紹介できる状態」と伝えている → 良い
- 一方で「人材紹介の件」が繰り返され、「お話をお伺いして」が先に出る → もう少しメリットを前に持ってくるとより良い

**After（さらに強化したイメージ）**

```
小山田：「お世話になっております。弊社、電気工事士や施工管理の20代・30代の方が複数名おりまして、
        御社の条件をお聞きして、合いそうな方を数名、メールでお送りできる状態です。
        ホームページの必要な資格をお持ちの方もいらっしゃいますので、
        もしよろしければ、勤務地や職種など、条件を少しお聞かせいただけますか？」
```

### シーン⑧ アメリカンエンジニアリング（小山田・実際）※接続後の訴求は良いケース

**小山田の実際の音声（担当者接続後）**

```
担当者：「あ、はい、ご連絡かりました。採用担当の金城と申します。」

小山田：「お手数になります。今回、人材紹介の件でご連絡させていただいたんですが。はい。
        弊社ですね、電気工事や施工管理の有資格者がですね、二、三十代の方多数動いておりまして、
        もしよろしければ、御社の条件に合った方を数名すぐにご紹介できる状態なんですが。
        はい、ありがとうございます。
        こちらでもしよろしければですね、募集要項等も少しお聞きしながら、メール等で職務経歴書や履歴書をですね、
        送らせていただいて、もし弊社のサービスでしたり、人材というところで興味あれば
        返信いただいてから打ち合わせさせていただければなというようなところだったんですが。」
```

**ポイント**
- 「有資格者が二、三十代多数」「数名すぐにご紹介できる状態」と訴求している → 良い
- 「人材紹介の件」が繰り返されるが、その後の訴求でカバーできている
- アメリカンは接続後の流れは比較的良い。開き方（受付突破）の改善が主な課題

## 3. 細部の確認：メールアドレス・担当者名

### シーン⑨ ゼニタカ組（小山田・実際）※メールアドレス誤りの事例

**実際に起きたこと**
- 送付先：`jinji@zenitaka.co.jp`（誤り）
- 正しい：`tky_jinji@zenitaka.co.jp`（TKY_が抜けていた）
- 結果：メールが届かず、フォローアップ架電で発覚

**改善のアウトプットイメージ**

```
送付前の確認フロー：
1. ホームページ等で正しいメールアドレスを事前確認（RA自身が裏で行う。復唱で十分）
2. 口頭で聞く際は「TKYアンダーバー人事」など、接頭辞を明確に復唱
3. 「tky_jinji@zenitaka.co.jp でよろしかったでしょうか」と最後に復唱して確認
```

### シーン⑩ 担当者名の聞き取り（茂野の良い例 vs 小山田の課題）

**茂野の実際（エアルエンジ）**

```
茂野：「何様ってお呼びしたらよろしいでしょうか」
相手：「すいません。早咲と申します」
茂野：「早咲様ですね。早咲様は、早いに先はどう書」
相手：「長咲の先で」
茂野：「長咲の先で。はい。早いはあのー、あれ」
相手：「早朝の」
茂野：「かしこまりました。」
```

**小山田の課題（ゼニタカ組）**
- 先日の架電で担当者名を聞きそびれた
- フォローアップ時に「名前を聞きそびれてしまいまして」と伝えている

**改善のアウトプットイメージ**

```
接続後・クロージング時に必ず：
「お電話ごちそうさまは何様でしょうか」
「漢字はどのようにお書きしますか」
→ その場でメモし、復唱して確認
```

## 5. まとめ：アウトプットイメージの確認ポイント

| 観点 | Before（小山田・実際） | After（改善イメージ） |
|------|------------------------|------------------------|
| **開き方** | 「人材紹介の件でご連絡したんですけれども」 | 「資格持ちで20〜30代が複数名、御社に合いそうな方をご紹介できれば」 |
| **受付の反応** | 社名・名前の確認が入りやすい | 用件が明確で、確認がスムーズ |
| **接続後** | 「採用ニーズをお聞きしたい」が先 | 「合いそうな方を数名お送りできる状態」を先に |
| **紹介料** | 相手が聞いてから答える | こちらから触れ、主導権を握る |
| **細部** | メールアドレス誤り、担当者名聞きそびれ | 事前確認・復唱・その場で聞き取り |
