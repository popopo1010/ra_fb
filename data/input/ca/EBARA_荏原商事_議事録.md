## 4. 採用要件・人物像

### 年齢目安

| 年齢 | 条件 |
|------|------|
| 20代〜30代 | 中心 |
| 東京支社の民間施工管理 | 40代まで検討可 |
| 正社員 | 上限目安は50歳前後 |
| 54歳以上 | 嘱託契約の相談となるケースあり（退職金制度の関係） |

### 資格要件

| 年代 | 要件 |
|------|------|
| 20代 | 資格なしでも検討可（施工管理経験、親和性のある業界経験があれば可） |
| 30代以上 | 公共工事ポジションは**1級または2級施工管理技士**（電気・管・土木いずれか）が必要 |

### 歓迎される経験

- 電気設備、管工事、ポンプ設備関連の施工管理経験
- 上下水道施設の運転管理・オペレーション経験
- 民間工事では職長経験者も検討可

### 敬遠されやすいケース

- 建築系（建物を建てる土木中心）のみの経験
- 業界親和性が低い完全異業種（飲食・接客のみ等）
