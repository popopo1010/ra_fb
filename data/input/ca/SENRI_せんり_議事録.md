## 1. 会社・事業概要

| 項目 | 内容 |
|------|------|
| 事業 | **RC造マンション**（3〜5階建て）を中心とした建築事業を展開 |
| 体制 | 電気設備工事および建築施工管理を自社で担う |
| 強み | グループ親会社が不動産の仕入れから販売まで一気通貫で行っており、**安定した案件供給**がある |

## 3. 募集職種・仕事内容

### 電気施工管理

- RC造マンションの電気設備工事に関する施工管理
- 1人で複数現場を担当するケースあり（経験に応じて3〜4棟程度）

### 建築施工管理

- RC造マンション新築工事の施工管理
- 電気と建築の選考基準に大きな差はなし（最終判断は各課長）

## 4. 採用ターゲット・人物要件

### 未経験者

| 項目 | 内容 |
|------|------|
| 年齢目安 | 20代前半 |
| 条件 | 短期離職がないこと |
| 重視 | 建築・ものづくりへの興味関心、体育会系でスピード感のある社風にフィットすること |

### 微経験〜経験者

| 項目 | 内容 |
|------|------|
| 年齢目安 | 30代まで |
| 資格 | 無資格・微経験でも検討可 |
| 経験 | 施工管理または現場経験があること |

### 共通して重視する点

- 明るくハキハキした受け答え
- 成長意欲が高いこと
- **他責思考でないこと**
- 社風への適応力

## 11. 今後の連携方針

1. 電気・建築ともに**継続的な人材紹介**を希望
