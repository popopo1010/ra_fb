| 資格 | 料率 |
|------|------|
| 第二種電気工事士、第三種電気主任技術者 | 35％ |

| 無資格・その他関連資格 | 35％ |

### 返金規定

| 期間 | 返還 |
|------|------|
| 30日以内 | 80％ |
| 60日以内 | 50％ |
| 90日以内 | 20％ |
| 180日以内 | 5％ |

### 先方要望

- **一律35％**での契約を希望（他社と同条件のため）
- 請求書発行後、**翌月末支払い**を希望
- → 料率および支払条件については、社内確認のうえ調整予定

### 理論年収の考え方

- 労働条件通知書に記載されている内容を基準
- 固定賞与・月給・職務手当は含める
- 実働に応じて変動する残業代は含めない
- 交通費については、含めない方向で要相談
