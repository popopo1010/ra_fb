# 法人面談議事録

過去の法人面談の議事録を格納するリファレンスフォルダです。

## 目的

**「どこまで聞けているか」のイメージをつかむため**

- 実際の面談で何が聞けているか
- どの程度の深さまでヒアリングできているか
- 初回面談チェックリストの項目が、実務ではどこまでカバーされているか

を把握するための参考資料として活用する。

## 使い方

- 新規RAのオンボーディング時に「聞ける範囲」のイメージ共有に使用
- 初回面談の準備時に、過去の議事録を参照してヒアリングの参考にする
- 初回面談チェックリストと照らし合わせて、聞けている項目・聞けていない項目を整理する

## 格納形式

- ファイル名: `{企業名略}_議事録_{日付}.md` など
- 内容: 面談日、参加者、聞けた項目、聞けなかった項目、メモ など
