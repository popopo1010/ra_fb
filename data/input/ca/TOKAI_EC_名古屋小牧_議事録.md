## 1. 募集職種・採用ポジション

| 項目 | 内容 |
|------|------|
| 募集職種 | **電気工事の施工管理職**に限定 |
| 備考 | 技術営業職ではなく、**現場を管理する施工管理**がピンポイント |
| 選考 | 電気工事経験・職務経歴書をもとに書類選考→面接可否判断 |

---

## 2. 書類選考の通過基準

| 要素 | 内容 |
|------|------|
| 判断要素 | 年齢、経験値、業界への意欲・チャレンジ精神を総合判断 |
| 35歳以上 | 経験不足の場合は通過が難しい |
| 経験豊富 | 年齢が高くても通過しやすい |
| 若手（20代） | 資格がなくても、**意欲があれば面接で判断**する方針 |

---

## 3. 未経験者・若手層の採用方針

| 項目 | 内容 |
|------|------|
| 年齢制限 | 未経験者に明確な年齢制限はなし |
| 年収 | 希望年収が社内スキル水準と比較されるため、若手・未経験者は**現実的な調整が必要** |
| 実例 | 20代前半・未経験で入社し、2年で2級電気工事施工管理技士を取得した社員が在籍 |
| 注意 | 未経験者は入社後数年間は年収が低めになる点を理解してもらう必要あり |
| ミスマッチ | 高年収を希望する応募者はミスマッチになりやすい |

---

## 4. ポテンシャル層（若手）の初年度年収

| 項目 | 内容 |
|------|------|
| 初年度年収 | 残業代込みで**400万円未満**（実態は**350万円弱**） |
| 賞与 | 「寸止めレベル」。中小企業として大企業並みの年収は提示できない |
| 想定残業 | 月平均20時間の時間外労働を想定した水準 |

---

## 5. 残業時間・労働時間管理

| 項目 | 内容 |
|------|------|
| 繁忙期 | 特に3月末まで残業が多くなる傾向 |
| 実態 | 物件によっては**月60〜80時間**の残業が発生するケースあり |
| 36協定 | 遵守 |
| 月45時間超 | 年6回まで |
| 単月100時間超 | 発生しないよう管理 |
| 安全委員会 | 設置し、労働時間管理を徹底 |
| 有給 | 全社員に対し、年5日以上の取得を実施 |

---

## 6. 経験者（施工管理）の採用基準・年収上限

| 項目 | 内容 |
|------|------|
| 年齢目安 | 30代〜40代前半が理想。40代後半も経験値次第で検討余地あり |
| 資格 | 1級電気工事施工管理技士などの有資格者は高く評価 |
| 事業拡大 | 人員が確保できれば拡大可能な状況 |
| 年収上限 | 中小企業のため、**年収600万円超は難しい** |
| マッチ | 高年収帯より、**若手〜中堅層の紹介がマッチしやすい**という認識で一致 |

---

## 7. 若手育成・キャリアパス

| 項目 | 内容 |
|------|------|
| 課題 | 20代〜30代の若手層が不足。育成を重視 |
| 育成 | OJT中心、先輩社員がマンツーマンで指導 |
| キャリア目標 | 3〜5年で、一定規模の工事を一人で管理できる状態を目指す |
| 昇進 | 一般職→主任→係長。賃金テーブル・賞与算定基礎賃金が役職で変動 |
| 昇格基準 | 年数固定ではなく、**資格取得・マネジメント力**が判断基準 |
| 大卒22歳入社 | 5年後に主任→さらに4年後に係長。努力次第で10年以内の係長昇格も可能 |

## 8. 拠点別の採用状況

| 拠点 | 採用方針・条件 |
|------|----------------|
| **名古屋** | 施工管理技士など**有資格者を優先** |
| **小牧** | 機械メーカー内での作業が中心。**若手であれば電気工事士資格なしでも書類選考対象** |
| 方針 | 「まず1人ずつ」進める |
| 外国人材 | 技能実習生、技人国の採用も進めている。最終判断は現場次第 |

## 10. 人材紹介条件（合意事項）

| 項目 | 内容 |
|------|------|
| 形態 | 完全成功報酬型 |
| 請求 | 候補者の入社が確定したタイミングで請求書を発行 |

## 11. 今後の進め方

1. **若手（ポテンシャル層）を中心**に候補者紹介を進める
2. 名古屋／小牧それぞれの**拠点特性を踏まえて**推薦
3. 紹介条件（料率35％）について最終確認後、契約締結
4. 継続的に情報共有を行いながら採用活動を推進
