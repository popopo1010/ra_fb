# データ

FB 生成の入出力データを格納。

## ディレクトリ構成

```
data/
├── input/              # 入力データ
│   ├── ra/             # 初回架電（RA）文字起こし
│   │   ├── 茂野/
│   │   ├── 小山田/
│   │   └── ...
│   └── ca/             # 法人面談（CA）議事録
│       ├── _template_議事録.md
│       └── {企業名}_議事録.md
└── output/
    └── 法人マスタ/      # 法人情報（FB生成時に自動格納）
        ├── マスタ項目一覧.md
        ├── _template_法人情報.md
        └── {会社名}.md
```

## データの流れ

| 種別 | 入力 | 出力 |
|------|------|------|
| RA | `input/ra/` | FB → Slack、`output/法人マスタ/` |
| CA | `input/ca/` | FB → Slack、`output/法人マスタ/` |
