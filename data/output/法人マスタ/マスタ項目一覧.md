# 法人マスタ 項目一覧

FB 生成時に文字起こしから抽出し、`data/output/法人マスタ/{会社名}.md` に格納する項目。

---

## 1. Frontmatter（YAML・必須）

| 項目 | 型 | 説明 | 例 |
|------|-----|------|-----|
| 会社名 | 文字列 | 法人名 | "東海EC" |
| 都道府県 | 配列 | 勤務地の都道府県。複数可 | ["愛知", "岐阜"] |
| セグメント | 配列 | 事業セグメント | ["電気系", "施工管理"] |
| 最終更新 | 文字列 | 更新日 | "2025-02-13" |
| 出典 | 文字列 | 初回架電 or 法人面談 | "初回架電" |

**セグメント候補**: 電気系、土木、建築、管工事、DC、再エネ、物流、工場、オフィス、公共、住宅

## 2. 法人情報（本体）

### 2.1 企業情報（統合セクション）

基本情報・事業リサーチ・企業スナップショットを1セクションに統合。重複を排除。

| 項目 | 説明 | 例 |
|------|------|-----|
| 会社名 | 正式名称・略称 | 東海EC |
| 都道府県 | 勤務地 | 愛知、岐阜 |
| 事業概要 | 主な事業・工事・案件、事業一覧・主軸事業 | 電気設備工事、施工管理、太陽光EPC。主軸：産業用太陽光 |
| マーケット成長性 | 成長度＋なぜ成長しているか（背景・要因）と競合 | 高。再エネ補助金・脱炭素需要で太陽光EPC拡大。EPC競合多数 |
| 法人HP URL | 公式サイト（Web検索で補足） | https://example.co.jp |
| 売上構成比 | 各事業の売上構成比（推定可） | 産業用太陽光 70%、住宅用 20%、その他 10% |
| 今後の注力領域 | 中期計画・IR・社長メッセージ等から推察 | 海外展開、リユース事業 |
| 企業スナップショット | 一言で言うとどういう会社か、業界内ポジション、直近の勢い | 東芝BPとして97年の電気商社。太陽光・セキュリティ新規展開中 |
| 前職規模別USP | 大手・中堅・零細出身者向けの一言USP | 大手：裁量と現場密着。中堅：成長ステージ。零細：安定基盤 |
| 採用意思決定者 | 人事部／現場部長／経営者直轄など | 現場部長が主導、人事がフォロー |
| 口コミ評価傾向 | OpenWork・転職会議等の評価傾向 | 評価3.5。成長実感は高、残業はやや多めの声 |

| 項目 | 説明 | 例 |

| 必要資格 | 必須・歓迎の資格 | 電気工事士、施工管理技士 |

### 2.4 労働条件

| 項目 | 説明 | 例 |
|------|------|-----|
| 出張 | 出張ありなし・頻度・範囲 | あり/なし、月1〜2回 |
| 残業時間 | 残業の状況 | 月20h程度、繁忙期多め |
| 休日・直行直帰・リモート | 働き方の詳細（1行に集約） | 完全週休2日、直行直帰OK、リモート不可 |
| 福利厚生 | 各種福利厚生 | 社会保険完備、資格手当、住宅手当 |

### 2.5 評価・キャリアアップ

| 項目 | 説明 | 例 |
|------|------|-----|
| 評価 | 人事評価制度、評価の仕方 | 年1回、目標管理制度 |
| キャリアアップ制度 | 昇格・昇給・キャリアパス | 資格取得支援、現場監督へのステップ |

### 2.6 面接・選考

| 項目 | 説明 | 例 |
|------|------|-----|
| 面接の方式 | オンライン/オフライン、形式 | 1次オンライン、2次対面 |
| 面接回数 | 選考の回数 | 2回、3回 |
| 面接で見ているポイント | 面接で重視している点 | コミュニケーション、現場経験 |
| 過去落とした人 | 落とした人の特徴・理由 | 志望動機が弱い、現場経験不足 |
| 受かった人 | 受かった人の特徴 | 即戦力、転職意欲が高い |

## 3. 候補者別訴求（統合セクション）

資格・経験軸と前職規模軸を1テーブルに統合。訴求・差別化は重複なく記載。

| セグメント | 採用 | 訴求・差別化 |
|------------|------|--------------|
| ①電気工事士もち、施工管理未経験 | ○/△/× | 未経験OK、資格取得支援、OJT 等 |
| ②施工管理資格もち、マネジメント未経験 | ○/△/× | 現場監督育成、段階的任せ 等 |
| ③施工管理資格もち、マネジメント経験あり | ○/△/× | 大型案件、裁量、キャリアパス 等 |
| 大手出身者向け（1,000名以上） | - | 一言USP |
| 中堅出身者向け（100〜999名） | - | 一言USP |
| 零細出身者向け（100名未満） | - | 一言USP |

都道府県×セグメントで比較する際の差別化ポイント。**他社との違いに特化**（企業の一般論は書かない）。
