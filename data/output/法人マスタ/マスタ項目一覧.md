# 法人マスタ 項目一覧

FB 生成時に文字起こしから抽出し、`data/output/法人マスタ/{会社名}.md` に格納する項目。

---

## 1. Frontmatter（YAML・必須）

比較の軸。都道府県×セグメントで絞り込みに使用。

| 項目 | 型 | 説明 | 例 |
|------|-----|------|-----|
| 会社名 | 文字列 | 法人名 | "東海EC" |
| 都道府県 | 配列 | 勤務地の都道府県。複数可 | ["愛知", "岐阜"] |
| セグメント | 配列 | 事業セグメント | ["電気系", "施工管理"] |
| 最終更新 | 文字列 | 更新日 | "2025-02-13" |
| 出典 | 文字列 | 初回架電 or 法人面談 | "初回架電" |

**セグメント候補**: 電気系、土木、建築、管工事、DC、再エネ、物流、工場、オフィス、公共、住宅

## 2. 法人情報（本体）

### 2.1 基本情報

| 項目 | 説明 | 例 |
|------|------|-----|
| 会社名 | 正式名称・略称 | 東海EC |
| 都道府県 | 勤務地 | 愛知、岐阜 |
| 会社の事業内容 | 主な事業・工事・案件 | 電気設備工事、施工管理、PM |
| マーケットの成長度合い | その会社がやっているマーケットの成長度。**なぜ成長しているか（背景・要因）**も記載 | 高。再エネ補助金・脱炭素需要で太陽光EPC拡大 |

### 2.1.1 事業リサーチ（Web検索で補足・アップロード時に自動取得）

文字起こしのみでは不足するため、会社名から Web 検索（IR・中期経営計画・社長メッセージ等）で補足。

| 項目 | 説明 | 例 |
|------|------|-----|
| 法人HP URL | 公式サイトURL（Web検索で取得） | https://example.co.jp |
| 事業一覧・主軸事業 | 展開している事業を一覧化し、主軸事業を特定 | 太陽光EPC、電気工事、部材販売。主軸：産業用太陽光 |
| 売上構成比 | 各事業の売上構成比（推定可） | 産業用太陽光 70%、住宅用 20%、その他 10% |
| 今後の注力領域 | 中期経営計画・IR・社長メッセージ等から推察 | 海外展開、リユース事業、独立電源 |
| マーケットの成長性・競合 | 事業領域の成長性（**なぜ成長しているか背景・要因を含む**）と競合状況 | 再エネ市場は成長中（補助金・脱炭素・FIT買取）。EPC競合多数 |

### 2.2 仕事内容

| 項目 | 説明 |
|------|------|
| 仕事内容 | 求人ポジションの具体的な業務内容 |

### 2.3 採用条件

| 項目 | 説明 | 例 |
|------|------|-----|
| 必要資格 | 必須・歓迎の資格 | 電気工事士、施工管理技士 |
| 必要経験 | 経験年数・内容 | 3年以上、現場経験 |
| 未経験OK | 未経験OKかどうか | ○/△/× |
| 年齢 | 年齢条件 | 特になし、20〜50代 |
| 採用人数 | 採用予定人数 | 3名、エリア別 |

### 2.4 労働条件

| 項目 | 説明 | 例 |
|------|------|-----|
| 出張 | 出張ありなし・頻度・範囲 | あり/なし、月1〜2回 |
| 残業時間 | 残業の状況 | 月20h程度、繁忙期多め |
| 福利厚生 | 各種福利厚生 | 社会保険完備、資格手当、住宅手当 |

### 2.5 評価・キャリアアップ

| 項目 | 説明 | 例 |
|------|------|-----|
| 評価 | 人事評価制度、評価の仕方 | 年1回、目標管理制度 |
| キャリアアップ制度 | 昇格・昇給・キャリアパス | 資格取得支援、現場監督へのステップ |

### 2.6 面接・選考

| 項目 | 説明 | 例 |
|------|------|-----|
| 面接の方式 | オンライン/オフライン、形式 | 1次オンライン、2次対面 |
| 面接回数 | 選考の回数 | 2回、3回 |
| 面接で見ているポイント | 面接で重視している点 | コミュニケーション、現場経験 |
| 過去落とした人 | 落とした人の特徴・理由 | 志望動機が弱い、現場経験不足 |
| 受かった人 | 受かった人の特徴 | 即戦力、転職意欲が高い |

### 2.7 その他

| 項目 | 説明 |
|------|------|
| 手数料 | 人材紹介手数料（聞けた場合） |
| 重視点 | 採用で重視している点 |

## 3. 候補者タイプ別

| 候補者タイプ | 採用 | 訴求・差別化 |
|--------------|------|--------------|
| ①電気工事士もち、電気施工管理未経験 | ○/△/× | 未経験OK、資格取得支援、OJT 等 |
| ②施工管理資格もち、マネジメント経験なし | ○/△/× | 現場監督育成、段階的任せ 等 |
| ③施工管理資格もち、マネジメント経験あり | ○/△/× | 大型案件、裁量、キャリアパス 等 |
